# assets/animations/

Place Lottie/Rive animation files here. Already mapped in pubspec.yaml
(`assets/animations/`), so anything dropped in this folder is available
immediately after `flutter pub get` — no pubspec edit needed.

Note: no animation package (e.g. `lottie`, `rive`) is included in Phase
1's dependencies, per the "no unnecessary packages" requirement — add
one when the first animation asset actually needs it.
